# Belch
*Poison | Special*

- **Accuracy:** Special + Channel
- **Damage:** Special + 5
- **Target:** Foe
- **Effect:** The user must eat something before using this Move. -1 Accuracy.

> The user lets out a horrible belch on the target. The air contains toxins that deal a terrible damage.
