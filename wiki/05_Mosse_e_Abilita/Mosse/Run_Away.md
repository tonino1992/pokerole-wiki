# Run Away
*Senza Tipo | Support*

- **Accuracy:** Dexterity + Athletic
- **Damage:** N/A
- **Target:** User
- **Effect:** L'Utente fugge dal campo di battaglia. Questo effetto pone fine alla battaglia. I nemici potrebbero tentare di impedirlo lanciando: Dexterity + Athletic e ottenendo gli stessi o più successi. Se l'utente è bloccato, questa manovra fallisce.
