# Charge
*Electric | Support*

- **Accuracy:** Insight + Nature
- **Damage:** N/A
- **Target:** User
- **Effect:** Add 2 Dice to the Damage Pool to the next Electric Damaging move the user performs.

> The user concentrates all the electricity in its body to be released.
