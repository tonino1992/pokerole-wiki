# Aura Wheel
*Electric | Physical*

- **Accuracy:** Dexterity + Brawl
- **Damage:** Strength + 4
- **Target:** Foe
- **Effect:** If successful, Increase user's Dexterity. If the user is in “Hangry” Form this Move is considered to be Dark-Type.

> The Pokemon happily runs into a wheel made of electricity that strikes the foe. If the user is feeling hungry the wheel's energy is replaced by a dark aura.
