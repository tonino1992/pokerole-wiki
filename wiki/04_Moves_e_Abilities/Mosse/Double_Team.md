# Double Team
*Normal | Support*

- **Accuracy:** Dexterity + Evasion
- **Damage:** N/A
- **Target:** User
- **Effect:** The User can now make up to 5 Evasions per Round.

> The user moves rapidly around the foe, creating illusory copies of itself.
