# Echoed Voice
*Normal | Special*

- **Accuracy:** Dexterity + Channel
- **Damage:** Special + 2
- **Target:** Foe
- **Effect:** Sound Based. This move can be used many times in the same Round. Add 1 Extra Dice to the Damage Pool for every consecutive time you use it. Up to 8 dice may be added this way. Reset the effect if the user performs another move, evasion, or clash.

> The user shouts loudly and the echo answers back, it will yell louder every time the echo talks back.
